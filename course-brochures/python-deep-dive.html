About This Course
What is Python Mastery — 90 Days?

Python Mastery is a comprehensive 90-day program built for people who want to go beyond tutorials and become genuinely job-ready. It's not a collection of videos — it's a structured learning experience with live sessions every weekend, weekly graded assignments, personal code reviews, and a capstone project you'll build from scratch and can add to your portfolio.


The program is designed around three months, each with a clear theme: Month 1 builds your core Python foundation deep enough to write professional code. Month 2 puts that foundation to work on real problems — APIs, databases, data analysis, automation. Month 3 goes professional: algorithms, version control, web APIs with FastAPI, Docker, and deployment — ending with a capstone project that demonstrates everything you've learned.


This is the program for you if you're serious. It asks for 8–10 hours per week. The people who get the most from it are those who show up to every session, do every assignment, and treat the code reviews as the learning opportunity they are. By Day 90 you'll have a portfolio, a network of peers, and the ability to walk into a junior Python developer interview with genuine confidence.

Curriculum
Three Months, Three Phases
Month 1
Core Python
  • Python fundamentals in depth — variables, types, control flow, functions, scope — at a pace that builds real understanding, not surface familiarity
  • Data structures & algorithms basics — lists, dicts, sets, and the algorithmic thinking needed to use them well
  • Functions, decorators & closures — advanced function features that professional Python code relies on; understanding Python's functional programming side
  • OOP — classes, inheritance, dunder methods — object-oriented programming done right; when to use classes and how to design them cleanly
  • Exception handling & logging — writing code that handles failure gracefully and produces meaningful diagnostic information
  • Unit testing with pytest — writing tests from day one; the habit that separates professional code from hobbyist code
Month 2
Applied Python
  • File & database I/O (SQLite) — working with files, CSVs, and a real relational database; CRUD operations with Python's sqlite3 module
  • REST APIs & web scraping — calling APIs with requests, parsing responses, and extracting data from web pages with BeautifulSoup
  • Data analysis with Pandas & NumPy — loading, cleaning, transforming, and analysing real datasets; the foundation of data science in Python
  • Automation scripts (OS, scheduling) — automating file operations, scheduling tasks with cron, building tools that run without you
  • Virtual environments & packaging — managing dependencies professionally; creating installable packages
  • Type hints & mypy — adding static type annotations to catch bugs before runtime; modern Python best practices
Month 3
Professional Python
  • Algorithm design & complexity — Big-O notation, common algorithm patterns, and how to reason about the performance of your code
  • Git & version control — collaborative workflows with GitHub; branching, pull requests, code review as practiced in real teams
  • Web APIs with FastAPI — building production-quality REST APIs with Python's fastest modern framework; authentication, validation, docs
  • Docker basics & deployment — containerising your applications and deploying them to the cloud; the complete path from local to live
  • Capstone project — build and ship a real project of your choice; mentored sprint with code reviews and a final demo
  • Portfolio review & interview prep — preparing your GitHub, resume, and ability to discuss your projects; mock interview practice
