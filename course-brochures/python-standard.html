About This Course
What is Python for Data Science — 8 Weeks?

Data science in Python has a well-defined core toolkit: NumPy for numerical computation, Pandas for data manipulation, and Matplotlib/Seaborn for visualisation. These three libraries, used well, let you take a raw dataset and produce real insights. This 8-week course teaches all three — thoroughly, with real datasets, in a way that builds lasting skills rather than surface familiarity.


The course is project-driven: every two weeks builds to a real dataset project. You'll clean a messy real-world dataset, run a full exploratory data analysis on Indian census data, and complete a capstone end-to-end analysis project of your choosing. By Week 8 you'll have a Jupyter notebook portfolio that demonstrates your ability to handle data from start to finish.


The weekend format (Saturday and Sunday mornings for 8 weeks) is designed to be sustainable alongside a full-time job or university. Sessions are two hours each, and recordings are available for anything you miss or want to review. Weekly assignments with real datasets ensure you're building fluency, not just following along.

Curriculum
Eight Weeks, Four Phases, Four Projects
Weeks 1–2
Python & NumPy Foundations
  • Python refresher (fast-paced) — a quick pass through the Python you'll use constantly in data science: list comprehensions, lambda functions, map/filter, and working with files
  • NumPy arrays & broadcasting — the ndarray as Python's most important data structure for numerical computing; array creation, indexing, slicing, and the broadcasting rules that let you write clean vectorised code
  • Vectorised operations — replacing Python loops with NumPy operations for 10–100x performance; universal functions (ufuncs) and when to reach for them
  • Random number generation & distributions — generating random data, sampling from distributions (uniform, normal, Binomial), and the applications in simulation and statistical analysis
Weeks 3–4
Pandas — Data Wrangling
  • DataFrames & Series — Pandas' two core data structures; creating them, understanding their index, and the relationship between them
  • Indexing, filtering & groupby — .loc vs. .iloc, boolean masking, and the groupby-apply-combine paradigm for aggregating data across categories
  • Merging, joining & reshaping — combining datasets with merge and join; pivoting and melting for reshaping data between wide and long formats
  • Handling missing data + Project — strategies for detecting, imputing, and removing missing values; Project: receive a genuinely messy real-world dataset and clean it into analysis-ready form
Weeks 5–6
Visualisation & EDA
  • Matplotlib & Seaborn — the two-layer approach to Python visualisation; Matplotlib for full control, Seaborn for statistical plots with less code; figure and axes anatomy
  • Exploratory Data Analysis workflow — the systematic process of getting to know a new dataset: distributions, outliers, correlations, and unexpected patterns
  • Distribution plots, heatmaps, pair plots — the visualisation types most useful for EDA; choosing the right chart for the question you're asking
  • Project: full EDA on Indian census data — a complete exploratory analysis on a real, meaningful dataset; the full EDA workflow from loading to conclusions
Weeks 7–8
Applied Analysis & Capstone
  • Statistical testing basics — hypothesis testing, p-values, t-tests, and chi-squared tests in Python; when to reach for a statistical test and how to interpret the result without being misled
  • Correlation vs. causation — the most important distinction in data analysis; how to present correlations honestly and what additional evidence is needed for causal claims
  • Reporting with Jupyter notebooks — structuring a notebook as a document someone else can read; markdown, clean visualisations, and the narrative arc of a data analysis report
  • Capstone: end-to-end analysis project — you choose a dataset on a topic you care about; full EDA, visualisations, statistical analysis, and a polished Jupyter notebook report with instructor feedback
