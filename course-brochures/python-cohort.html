About This Course
What is the Python Weekend Intensive?

The Python Weekend Intensive is a two-day, 16-hour live cohort designed to take you from zero programming experience to building three real Python projects — all in a single weekend. No previous coding knowledge is assumed, and no prior preparation is required.


Unlike self-paced video courses where you're alone with a screen, this is a small-group live experience. You'll work alongside a cohort of peers, get immediate feedback after every exercise block, and have direct access to your instructor throughout. The format is structured around rapid cycles of explanation, hands-on coding, and review — the same rhythm used in the best bootcamps, compressed into a weekend.


By Saturday evening you'll have the foundations down. By Sunday afternoon you'll have built a data parser, a live weather CLI app, and a file automation script — all projects you can show to others and learn from. You'll also leave with a personalised roadmap for what to tackle next based on your goals.

Is This Right for You?
Who Should Attend
✓ This is for you if…
  • You're an absolute beginner who wants to go from zero to building real things — fast
  • You've tried learning Python online but keep losing momentum with self-paced content
  • You prefer learning in a live, interactive environment where you can ask questions

Curriculum
Two Days, Four Blocks
Day 1 — Morning
Python Foundations
  • Setup & environment — installing Python, setting up VS Code, running your first line; we handle everyone's setup issues live
  • Variables, operators, data types — how Python represents and manipulates information; int, float, string, bool in practice
  • Control flow & loops — if/elif/else, for and while loops; building the logic backbone of any program
  • Functions & scope — defining reusable code blocks, understanding how Python manages variable scope
Day 1 — Afternoon
Data Structures + Project 1
  • Lists, tuples, dicts, sets — Python's four data containers with hands-on practice for each; when to reach for which
  • List comprehensions — the Pythonic way to transform and filter collections in one clean expression
  • Nested data structures — working with real-world data shapes: lists of dicts, dicts of lists
  • Project 1: Data parser & cleaner — read a messy CSV, clean it up, and produce a structured output; your first real Python program
Day 2 — Morning
Intermediate Python
  • Object-Oriented Programming — classes, instances, methods, and inheritance; understanding the OOP mental model and when it's useful
  • File I/O & CSV handling — reading, writing, and processing files; working with the csv module for structured data
  • Error handling & logging — writing code that fails gracefully; try/except patterns and using the logging module for debugging
  • Modules, packages & pip — how Python's import system works; installing and using third-party libraries
Day 2 — Afternoon
Applied Projects + Roadmap
  • Calling REST APIs — making HTTP requests, parsing JSON responses, handling errors; the pattern behind every modern data integration
  • Project 2: Live weather CLI app — build a command-line tool that fetches real weather data for any city using a free API
  • Project 3: File automation script — build a script that organises files in a folder automatically; a taste of real-world automation
  • Code review & personal roadmap — instructor reviews each project; you leave with a tailored plan for what to learn next based on your goals
